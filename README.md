# Express Task Manager

A simple task manager built with Node.js, Express, EJS, and the File System module.

## Features

- Create tasks
- View all tasks
- Read task details
- Edit task titles
- Store tasks as text files
- Server-side rendering using EJS

## Tech Stack

- Node.js
- Express.js
- EJS
- Tailwind CSS
- File System (fs)

## Installation

1. Clone the repository

```bash
git clone https://github.com/priteshgholap2004/express-task-manager.git
```

2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
node index.js
```

or

```bash
npx nodemon index.js
```

4. Open in browser

```
http://localhost:3000
```
